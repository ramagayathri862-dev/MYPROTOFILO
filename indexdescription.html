About This Webpage

  1. This page is created as the About page of my personal portfolio website.
  2. The HTML document structure is created using the basic HTML tags such as html, head, and body.
  3. The title tag is used to display the page title on the browser tab.
  4. An external CSS file named websitestyle.css is linked to style the webpage.
  5. The body tag contains a class called about-page which helps apply specific CSS styles to this page.
  6. A navigation bar is created using the nav tag with a link that allows users to go back to the Home page.
  7. A heading tag (h2) is used to display the title "About This Webpage".
  8. An ordered list (ol) is used to show the description points in a numbered format.
  9. Each description point is written using list item (li) tags.
  10. An anchor tag (a) with a button is used to create a clickable button that redirects to the Home page.
  11. A heading (h1) is used to introduce the code screenshot section.
  12. An image is displayed using the img tag to show the screenshot of the portfolio code.
  13. This page helps explain how the portfolio homepage was designed using HTML and CSS.
